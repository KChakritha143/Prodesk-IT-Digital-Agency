# Prodesk-IT-Digital-Agency
Prodesk IT Digital Agency landing page built using HTML, CSS (Flexbox/Grid), and JavaScript. This project focuses on responsive design, pixel-perfect layouts, dark mode toggle, and interactive UI elements, simulating a real-world frontend development task.

# Project Objective
The main goal and purpose of this project was to improve my frontend skills by focusing on:
- Working with Flexbox and CSS Grid
- Creating responsive layouts for different screen sizes
- Practicing basic javascript
- Building a clean and professional UI

# Features
## Core Features(Level-01):
- A responsive navigation bar that works on both desktop and mobile consists of logo, links         (Home,About,Services and Contact) on the right.On mobile,there be a hamburger menu.
- A hero section with a catchy headline, subtext, and a call-to-action (Get Started) button.
- A services section consists of services like SEO,Web development and Marketing designed using     Flexbox/Grid 
- A simple footer with Copyright text and social media icons
## Intermediate Features(Level-02):
- Dark mode toggle that switches the entire site between Light mode and Dark mode.
- Smooth hover effects on buttons and service cards
- A sticky navbar that stays visible while scrolling

# Technologies Used
- HTML5 for structure
- CSS3 for styling
- Javascript for interactivity

# Constraints followed
- I did not use bootstrap
- I did not use Tailwind CSS
- All styling is done using raw CSS

# How to run
1. Download the repository
2. Open the "index.html" file in browser.

# What I learned through this process
- How to structure a real-world landing page  
- Better understanding of CSS layouts and positioning  
- How to make a website responsive  
- Adding simple interactivity using JavaScript
